# RAG Knowledge Base

This directory is the Markdown-first research knowledge layer for the Vibe Research Pipeline.

## Entry points

- [Index](index.md)
- [Vocabulary](vocabulary.md)
- [Source template](template.md)
- [Operation log](log.md)
- [References manifest](references.bib)

## Layout

- `reference/` stores raw materials such as PDFs, BibTeX, Zotero exports, and import artifacts.
- `summary/sources/` stores one structured source page per paper or research material.
- `summary/synthesis/` stores cross-source synthesis, consensus, disputes, and note-derived knowledge.
- `summary/<dimension>/` directories are generated by initialization from project-specific dimensions.

Use the skills in `.claude/skills/rag*` for operations. Keep project knowledge in this directory, not in `CLAUDE.md`.
